ELEVENLABS WEBHOOK CONFIGURATION
=================================

URL: https://refreshstudio-production.up.railway.app/api/booking/webhook/elevenlabs-unified
Method: POST

Parameters:
- action (required): get_available_times, get_soonest_available, book_appointment, cancel_appointment
- date (optional): Date in DD.MM.YYYY format
- time (optional): Time in HH:MM format  
- customer (optional): JSON object with firstName, lastName, email, phone
- phone (optional): Phone number for cancellation

Example customer object:
{
  "firstName": "Mária", 
  "lastName": "Novák",
  "email": "maria@example.sk", 
  "phone": "+421901234567"
}

Actions:
1. get_available_times - needs: action, date
2. get_soonest_available - needs: action only
3. book_appointment - needs: action, date, time, customer
4. cancel_appointment - needs: action, phone
